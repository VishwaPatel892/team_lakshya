# 🚀 LAKSHYA

> **LAKSHYA** is a next-generation **AI Browser Companion** that transforms the way users browse, learn, analyze, and interact with web content.

Unlike traditional AI chatbots that require users to **copy and paste content**, LAKSHYA works **directly inside the browser**, understands the current webpage, and provides contextual AI assistance without interrupting the browsing experience.

It acts as a unified AI workspace for **students, developers, researchers, professionals, and everyday users**, bringing multiple AI capabilities together in one intelligent browser extension.

---

# 🌟 Inspiration

Modern AI tools are incredibly powerful, but they still require users to constantly switch between applications.

A typical workflow today looks like this:

```text
Read Website
      │
      ▼
Copy Content
      │
      ▼
Open AI Chatbot
      │
      ▼
Paste Content
      │
      ▼
Ask Question
      │
      ▼
Return to Website
```

This repetitive process breaks focus, wastes time, and reduces productivity.

We wanted to eliminate this friction completely.

# 📦 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/lakshya.git
```

Navigate to the project

```bash
cd lakshya
```

Install dependencies

```bash
npm install
```

Run Frontend

```bash
npm run dev
```

Run Backend

```bash
npm run server
```

Build Extension

```bash
npm run build
```

2. Navigate to

